# Guida Utente — Pannello di Amministrazione

Questa guida spiega come gestire i contenuti del sito in autonomia: notizie, progetti, servizi e impostazioni generali. Non è necessaria nessuna competenza tecnica — basta seguire i passaggi descritti qui sotto.

Per qualsiasi problema non coperto da questa guida, contatta lo sviluppatore.

---

## Accesso al Pannello Admin

**Indirizzo:** `https://tuodominio.com/it/admin/`

### Come accedere

1. Apri il link qui sopra nel browser.
2. Clicca su **"Login with GitHub"**.
3. Se è la prima volta, GitHub chiederà di autorizzare l'applicazione — clicca **"Authorize"**.
4. Una volta autorizzato, verrai reindirizzato automaticamente al pannello di amministrazione.

> **Nota:** Usa **Google Chrome** o **Microsoft Edge**. Firefox non è supportato e potrebbe causare problemi di caricamento o di salvataggio.

## Notizie

La sezione **Notizie** ti permette di pubblicare aggiornamenti, comunicati e articoli sul sito.

### Aggiungere una notizia

1. Nel menu laterale, clicca su **"Notizie / News"**.
2. Clicca il pulsante **"Nuovo"** (o **"New"**) in alto a destra.
3. Compila i campi:
   - **Titolo (IT)** — Il titolo dell'articolo in italiano. Campo obbligatorio.
   - **Titolo (EN)** — La versione inglese del titolo. Puoi lasciarlo vuoto: verrà **tradotto automaticamente** al salvataggio.
   - **Data pubblicazione** — La data in cui la notizia viene visualizzata sul sito.
   - **Descrizione breve (IT)** — Un riassunto di una riga, visibile nelle anteprime. Campo obbligatorio.
   - **Descrizione breve (EN)** — Traduzione automatica se lasciata vuota.
   - **Immagine copertina** — Foto principale dell'articolo. Dimensione consigliata: **1200×480 px**. Campo facoltativo.
   - **Contenuto (IT)** — Il corpo dell'articolo, con supporto per testo in grassetto, corsivo, elenchi e link.
   - **Contenuto (EN)** — Traduzione automatica se lasciato vuoto.
4. Clicca **"Salva"** (o **"Save"**) in alto a destra.

> **Traduzione automatica:** Quando il sito è impostato in modalità bilingue, i campi in inglese contrassegnati con "auto-tradotto al salvataggio" vengono compilati automaticamente se li lasci vuoti. Se lasci il campo vuoto, la traduzione viene generata automaticamente al salvataggio. Puoi comunque modificare manualmente il testo tradotto in seguito. Oppure puoi scriverli a mano se preferisci una traduzione personalizzata fin dall'inizio.

### Modificare una notizia

1. Nella lista delle notizie, clicca sul titolo di quella che vuoi modificare.
2. Apporta le modifiche desiderate.
3. Clicca **"Salva"**.

### Eliminare una notizia

1. Apri la notizia che vuoi eliminare.
2. Clicca il menu con i tre puntini (**⋮**) o il pulsante **"Elimina"** (dipende dalla versione del pannello).
3. Conferma l'eliminazione.

## Progetti

La sezione **Progetti** raccoglie i lavori svolti dallo studio, ciascuno con una scheda dettagliata.

### Aggiungere un progetto

1. Nel menu laterale, clicca su **"Progetti / Projects"**.
2. Clicca **"Nuovo"**.
3. Compila i campi:

   | Campo | Descrizione |
   |---|---|
   | **Titolo (IT)** | Nome del progetto in italiano. Obbligatorio. |
   | **Titolo (EN)** | Tradotto automaticamente se vuoto. |
   | **Anno** | Anno di realizzazione (es: `2023`). |
   | **Categoria (IT)** | Tipo di intervento (es: `Progettazione strutturale`). Tradotto automaticamente se vuoto. |
   | **Località (IT)** | Dove si trova il progetto (es: `Napoli`). Tradotto automaticamente se vuoto. |
   | **Ruolo (IT)** | Il ruolo dello studio nel progetto (es: `Progettista`). Tradotto automaticamente se vuoto. |
   | **Stato (IT)** | Stato del progetto (es: `Completato`, `In corso`). Default: `Completato`. |
   | **Descrizione breve (IT)** | Testo introduttivo visualizzato in cima alla pagina progetto. Tradotto automaticamente se vuoto. |
   | **Risultato (IT)** | Un breve testo che evidenzia il risultato raggiunto. Appare in un box separato. Tradotto automaticamente se vuoto. |
   | **Descrizione completa** | Testo esteso con formattazione avanzata (titoli, elenchi, link). Campo facoltativo. |
   | **Immagine principale** | Foto principale del progetto. Dimensione consigliata: **1200×800 px**. Obbligatoria. |
   | **Galleria foto** | Tutte le foto aggiuntive del progetto, mostrate nel carosello. |

4. Clicca **"Salva"**.

### Come aggiungere foto alla galleria

1. Scorri verso il basso fino al campo **"Galleria foto"**.
2. Clicca **"Aggiungi"** (o il pulsante **+**).
3. Carica o seleziona una foto dalla libreria media.
4. Ripeti per ogni foto che vuoi aggiungere.
5. Puoi trascinare le foto per riordinarle.
6. Salva il progetto.

> **Dimensione consigliata per le foto galleria:** 1600×1067 px (proporzione 3:2).

### Modificare o eliminare un progetto

Stesse istruzioni della sezione Notizie: apri il progetto dalla lista, modifica i campi e salva — oppure usa il pulsante elimina.

## Servizi

La sezione **Servizi** (nel pannello chiamata **"Servizi / Services"**) contiene le soluzioni offerte dallo studio.

### Aggiungere un servizio

1. Nel menu laterale, clicca su **"Servizi / Services"**.
2. Clicca **"Nuovo"**.
3. Compila i campi:

   | Campo | Descrizione |
   |---|---|
   | **Titolo (IT)** | Nome del servizio in italiano. Obbligatorio. |
   | **Titolo (EN)** | Tradotto automaticamente se vuoto. |
   | **Descrizione breve (IT)** | Una sola riga. Mostrata nelle card e nella lista servizi. Tradotta automaticamente se vuota. |
   | **Icona (emoji)** | Una emoji che rappresenta il servizio (es: `🏛️`, `⚙️`, `📐`). |
   | **Ordine di visualizzazione** | Numero intero che determina la posizione nella lista (1 = primo). |
   | **Immagine copertina** | Foto principale. Dimensione consigliata: **1200×800 px**. Facoltativa. |
   | **Contenuto (IT)** | Descrizione completa del servizio con formattazione avanzata. Tradotto automaticamente se vuoto. |
   | **Galleria foto** | Foto mostrate nel carosello nella pagina del servizio. Facoltativa. |

4. Clicca **"Salva"**.

### Modificare l'ordine dei servizi

Per cambiare l'ordine in cui i servizi appaiono sul sito, modifica il campo **"Ordine di visualizzazione"** di ciascun servizio: il valore `1` appare per primo, `2` per secondo, e così via.

### Modificare o eliminare un servizio

Come per le altre sezioni: apri dalla lista, modifica, salva — o elimina.

## Impostazioni Sito

La sezione **"Impostazioni Sito"** contiene i dati generali dello studio e le configurazioni globali del sito. Si trova nel menu laterale con l'icona ⚙️.

Clicca su **"Generali"** per aprire il modulo.

| Campo | Descrizione |
|---|---|
| **Immagine banner CTA homepage** | Foto di sfondo per il box "Hai un progetto in mente?" in homepage. Dimensione consigliata: 1920×600 px. |
| **Formspree ID** | Codice identificativo del modulo di contatto (es: `mlgobndz`). Necessario per ricevere email dal sito. Registrati gratuitamente su formspree.io — il codice appare nell'URL del form dopo `/f/` (es: `https://formspree.io/f/mlgobndz`). |
| **Logo sito** | Il logo dello studio (consigliato: PNG con sfondo trasparente). Se vuoto, viene usato il logo predefinito. |
| **File CV (PDF)** | Il curriculum vitae in formato PDF, scaricabile dai visitatori del sito. |
| **Tema predefinito del sito** | Il tema grafico mostrato ai nuovi visitatori. Opzioni disponibili: Steel, Blueprint, Obsidian, Industrial, Forest, Limestone, Marble, Daylight, High Contrast. |
| **Mostra selettore tema** | Se attivo, i visitatori possono scegliere il loro tema preferito. Se disattivato, viene usato solo il tema predefinito. |
| **Lingua del sito** | Modalità bilingue (IT + EN), solo italiano o solo inglese. Determina anche se la traduzione automatica è attiva. |
| **Cloudflare Analytics Token** | Token per il tracciamento delle visite (opzionale). Lascia vuoto per disabilitare le statistiche. |
| **Email** | Indirizzo email di contatto, visibile nel footer e nella pagina contatti. |
| **Telefono fisso** | Numero di telefono fisso (es: `+39 081-1808-8820`). |
| **Cellulare** | Numero di cellulare (es: `+39 333-40-46-355`). |
| **Indirizzo** | Indirizzo completo dello studio. |
| **Partita IVA** | Solo il numero, senza la sigla "P.IVA:" davanti. |
| **Orari** | Orari di apertura (es: `Lun - Sab: 8:00 - 19:00`). |
| **URL LinkedIn** | Indirizzo completo del profilo LinkedIn dello studio. |

Dopo aver modificato uno o più campi, clicca **"Salva"** in alto a destra.

## Strumento Cache CDN

**Indirizzo:** `https://tuodominio.com/it/admin/cache`

Puoi aprire questo strumento anche dall'interno del pannello admin, tramite il pulsante **"Cache CDN"** visibile in basso a destra.

### Perché esiste questo strumento

Le immagini del sito vengono servite tramite Cloudinary, un servizio di ottimizzazione e distribuzione delle immagini. Quando un file viene caricato per la prima volta, Cloudinary lo memorizza nelle sue cache globali per garantire tempi di caricamento veloci in tutto il mondo. Se sostituisci un file con lo stesso nome, la cache non si aggiorna da sola automaticamente — questo strumento forza il rinnovo manuale.

### Quando usarlo

Questo strumento serve **soltanto** quando hai sostituito un'immagine mantenendo lo stesso nome file. In quel caso, la CDN (il sistema di distribuzione delle immagini) potrebbe continuare a mostrare la vecchia versione per qualche ora.

Per aggiornamenti normali — aggiungere nuove immagini o immagini con nomi diversi — non è necessario usare questo strumento.

### Come usarlo

1. Apri la pagina della cache (`/it/admin/cache`).
2. Vedrai una griglia con tutte le immagini del sito, organizzate per sezione (Sito, Servizi, Progetti, Notizie).
3. Clicca sulle immagini che vuoi aggiornare per selezionarle (un segno di spunta apparirà sull'immagine).
4. Puoi usare il pulsante **"Seleziona tutto"** se vuoi aggiornare tutte le immagini in una volta.
5. Clicca il pulsante **"Invalida cache"**.
6. Aspetta qualche secondo: apparirà un riepilogo con i risultati dell'operazione.

## Domande Frequenti

**Ho salvato una modifica ma il sito non è cambiato — cosa è successo?**

È normale. Dopo ogni salvataggio, il sito viene ricostruito automaticamente. Il processo richiede in genere **1-3 minuti**. Aggiorna la pagina del sito dopo qualche minuto e le modifiche saranno visibili.

**Ho eliminato un contenuto per sbaglio — posso recuperarlo?**

Sì. Contatta lo sviluppatore: ogni modifica viene salvata nella cronologia git del progetto, e qualsiasi contenuto può essere ripristinato.

**Quali formati di immagine posso caricare?**

Il pannello accetta immagini in formato **JPG**, **PNG** e **WebP**. Per i loghi è consigliato il PNG con sfondo trasparente. Per i CV e documenti, usa il formato **PDF**.

**Il pannello admin non si apre o si comporta in modo strano.**

Verifica di stare usando **Google Chrome** o **Microsoft Edge**. Firefox non è supportato. Se il problema persiste anche su Chrome, prova a svuotare la cache del browser (tasto F5 o Ctrl+Shift+R) oppure ad aprire una finestra in modalità anonima.

**Posso aggiungere una nuova pagina al sito?**

No, l'aggiunta di nuove pagine richiede l'intervento dello sviluppatore. Il pannello admin gestisce solo i contenuti delle pagine già esistenti.

**Posso modificare i testi fissi del sito (intestazioni, menu, footer)?**

I testi principali delle pagine (come la sezione "Chi Siamo" e la pagina "Prezzi") si modificano tramite la sezione **"Contenuto Pagine"** nel menu laterale. Testi come le voci del menu di navigazione richiedono invece l'intervento dello sviluppatore.
